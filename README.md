# rag_agent
RAG agent based off LangChain and Unite.AI's tutorials.

### Setup
create virtual env
```sh
python3.12 -m env rag_agent_env
```
activate virtual env, then install requirements using make
```sh
source rag_agent_env/bin/activate
make dev_env
```
run script
```sh
python3.12 rag_agent.py
```
